GENAI - 圖像生成系統

🧠 作品敘述

本作品示範如何透過 Python 與 OpenAI API 結合 DALL·E 模型，進行文字轉圖像的生成任務。使用者輸入提示詞（Prompt），即可取得對應的圖像網址，適用於 AI 藝術創作、生成式設計與創意應用展示。

🔍 功能介紹

  • ✅ 結合 dotenv 安全讀取 OpenAI API 金鑰
  • ✅ 使用 OpenAI Python SDK 呼叫 DALL·E-3 圖像生成模型
  • ✅ 提供文字 prompt，自動回傳圖像網址
  • ✅ 適用於生成創意視覺內容或快速圖片原型

💻 程式碼展示

#Filename: W606_GenAI_ChatGPT_QuickStart_03.py
import datetime

from openai import OpenAI
import os
import openai

from dotenv import load_dotenv
load_dotenv(override=True)

openai.api_key = os.getenv('OPENAI_API_KEY')
client = OpenAI(api_key= openai.api_key )

response = client.images.generate(
    model="dall-e-3",
    prompt="a white siamese cat",
    size="1024x1024",
    quality="standard",
    n=1,
)

print(response.data[0].url)

print("\nW6 Assignment 3 | B11108037 |", datetime.datetime.now())
